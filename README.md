# LMS | Online Academy

Learning Management System (LMS) robustly built with clean architecture standards.

## Tech Stack
- **Python 3.14+**
- **Django 6.x**
- **Poetry** (Dependency management)
- **Architecture**: Lightweight Domain-Driven Design (DDD)

## Initial Setup
1. Clone the repository.
2. Install dependencies: `poetry install`.
3. Create a `.env` file based on `.env.example`.
4. Run migrations: `python manage.py migrate`.
